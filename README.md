## 🚀 Featured Projects

### 🐾 [PetAdopt - Pet Adoption Platform](https://github.com/musa195420/PetAdoption)

A cross-platform pet adoption ecosystem built with **Flutter** and a **Next.js / Express.js** backend.

**Key Features:**

- Role-based system for Adopters, Donors, and Admins
- ML-powered pet species detection using a PyTorch Lite model
- Real-time chat between adopters and donors using Firebase Firestore
- Location sharing and verification for safer adoption meetups
- Pet health tracking with medical history and vaccination records
- Advanced filtering by species, breed, age, and location

**Tech Stack:**

- Flutter with MVVM architecture
- Next.js and Express.js backend
- PostgreSQL with Supabase
- JWT authentication with refresh token flow
- Supabase storage with Multer integration
- PyTorch Lite image classification

[📱 View Mobile App](https://github.com/musa195420/PetAdoptionApp) | [🔧 View Backend API](https://github.com/musa195420/PetAdoption)

### 🎯 [Lumi H5P - Flutter H5P Player](https://pub.dev/packages/lumi_h5p)

A Flutter package for playing **offline H5P interactive content**, useful for educational and e-learning apps.

**Key Features:**

- Play `.h5p` files offline after initial download
- No H5P server required
- Auto-download support from direct URLs
- Progress tracking with real-time download indicators
- Cross-platform support for iOS, Android, and Web

**Tech Stack:**

- Flutter WebView integration
- Local file system management
- HTTP and HTTPS file handling
- Custom network security configurations
